## Features

- Automatically sync SSH & API keys
- Secure storage of credentials
- Easy-to-use CLI

## Requirements

- Python 3.7+
- [pip](https://pip.pypa.io/en/stable/)
- Scaleway Python SDK

## Installation

```bash
git clone https://github.com/SeeMyPing/scw-key-manager.git
cd scw-key-manager
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
